# DevDiary 📝

**DevDiary** is a modern developer-focused blogging platform where developers can write, share, and manage their technical thoughts, tutorials, and learning experiences.

This project is built with a modern full-stack JavaScript architecture using **React** for the frontend and **Appwrite** as the backend service. The UI is styled with **Tailwind CSS** and enhanced with **shadcn/ui** components to provide a clean, responsive, and accessible user interface.

## 🚀 Features

* ✍️ Create, edit, and delete blog posts
* 🔐 Authentication and user management with Appwrite
* 📂 Organized developer diary for sharing knowledge
* 🎨 Clean and responsive UI using Tailwind CSS & shadcn/ui
* ⚡ Fast and modern React-based frontend

## 🛠 Tech Stack

* **Frontend:** React
* **Backend:** Appwrite
* **Styling:** Tailwind CSS
* **UI Components:** shadcn/ui

## 📌 Purpose

DevDiary is designed as a personal developer blog platform where developers can document their learning journey, share technical insights, and build in public.
